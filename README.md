# Ganjil Genap #

## Deskripsi Program
Program GanjilGenap digunakan untuk menentukan apakah suatu bilangan yang dimasukkan oleh pengguna merupakan bilangan ganjil atau bilangan genap.
Program ini memanfaatkan operasi modulus (%) untuk mengecek sisa hasil bagi bilangan terhadap 2.

## Cara Kerja Program

# Input Data dari Pengguna
Program meminta pengguna untuk memasukkan sebuah angka melalui keyboard menggunakan kelas Scanner.

# Proses Pengecekan Bilangan
-Program memeriksa apakah bilangan tersebut habis dibagi 2 menggunakan operator modulus (%):
if (angka % 2 == 0)
-Jika hasilnya 0, maka bilangan tersebut genap.
-Jika hasilnya bukan 0, maka bilangan tersebut ganjil.

# Output yang Ditampilkan
Program akan menampilkan hasil berupa teks:
-“<angka> adalah angka bilangan genap”
  atau
“<angka> adalah angka bilangan ganjil”

## Contoh Input dan Output

# Input:
masukkan angka: 7

# Output:
7 adalah angka bilangan ganjil.

# Input:
masukkan angka: 12

# Output:
12 adalah angka bilangan genap
